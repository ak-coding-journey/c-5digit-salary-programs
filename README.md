# C 5-Digit Number & Salary Programs

This repository contains a **single C program** that performs multiple tasks:

1. Calculate **Gross Salary** (Basic + DA + HRA)  
2. Sum of all digits of a 5-digit number  
3. Sum of 1st, 3rd, and last digits of a 5-digit number  
4. Sum of 1st and last digits of a 5-digit number  
5. Reverse a 5-digit number  

The program runs **all tasks sequentially** with clear separation for readability.
